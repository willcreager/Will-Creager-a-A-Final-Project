# Schema Information

column name | data type | details
------------|-----------|-----------------------
id          | integer   | not null, primary key
author_id   | integer   | not null, foreign key (references users)
team_tag_id | integer   | not null, foreign key (references team tags)
title       | string    | not null

## followings
column name | data type | details
------------|-----------|-----------------------
